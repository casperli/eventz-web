Generators

gulp component --name [componentname]

Adds a new Angular component (including tests, styles, and a service) into the app folder. 

gulp component --name [componentname] --noservice

Adds a new Angular component - same as above but without the service. 

gulp component --name [componentname] --parent [parentcomponent]

Adds a new Angular component to a parent component. It is put into a sub folder of the parent component folder.

Task Runner

npm start

Compiles and injects all TypeScript files into the index.html, compiles all LESS files into one CSS file and starts an HTTP server that refreshes automatically when files change.

Building

npm run build

Compiles, minifies, and injects all TypeScript files into the index.html and compiles all LESS files into one CSS file. This generates an app for distribution.

Testing

npm test

Runs all your jasmine/karma unit tests. 

npm run e2e

Runs all your protractor end-to-end tests.
